# Sparkle

Sparkle is an educational C++ library for experimenting with windowing, rendering, widgets, and supporting utilities.

The API documentation is intentionally configured to become strict over time. When adding or updating public classes, document:

- the purpose of the class;
- how to use the most common command or method;
- a short C++ example;
- related classes in a `@see` section.


See [Sparkle API documentation guidelines](documentation_guidelines.md) for the full class-comment checklist and template.
